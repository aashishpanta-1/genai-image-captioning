This is my first project in python and it is about Generative AI auto image captioning. This project uses a pre-trained model to generate captions for images. The model is trained on a large dataset of images and their corresponding captions. The project is implemented using Python and the TensorFlow library. The code is organized into several modules, including data preprocessing, model training, and caption generation. The project also includes a user interface that allows users to upload images and receive generated captions in real-time.
